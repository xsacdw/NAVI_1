# 🍀 Lucky Lotto

> 버튼 하나로 로또 5게임 번호를 한번에 추천받는 웹사이트

## 기능
- 🎱 5게임 번호 동시 생성
- 🌙 / ☀️ 다크 / 라이트 모드 토글
- 🎨 번호 대역별 색상 구분 (1~10: 노랑, 11~20: 파랑, 21~30: 빨강, 31~40: 회색, 41~45: 초록)
- 💾 테마 설정 자동 저장 (localStorage)
